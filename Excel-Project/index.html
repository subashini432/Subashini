Excel Sales Analysis Project

This project demonstrates advanced Excel skills through a complete sales data analysis workflow including data preparation, formulas, dashboards, and VBA automation.

Dataset Used

  • Superstore Sales Dataset
  • Orders, Returns, and Users reference tables

Raw Data Preparation

Raw sales data was cleaned, structured, and prepared as a fact table for analysis. Duplicates were removed and column formats were standardized.

Manager Mapping (XLOOKUP)

Manager names were mapped to each order based on Region using XLOOKUP.

Return Status Mapping

Return status was derived by mapping Order IDs with the Returns table using XLOOKUP and logical functions.
